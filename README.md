# SQL Learning Hub — Interactive SQL Learning Platform
## Overview

This project is a database management system-based web application developed as part of our DBMS coursework. The main objective was to design and implement the core database concepts learned in the course.

Additionally, the application aims to make SQL concepts easier to understand for students by providing simple visualizations and an interactive interface.

The application allows users to interact with the database through a
simple web interface and perform operations like storing, retrieving,
and managing data efficiently.

## Features

-   User-friendly web interface
-   Database integration for storing and retrieving data
-   CRUD operations (Create, Read, Update, Delete)
-   Structured schema design using DBMS concepts
-   Real-time interaction with backend


## Technologies Used

-   Frontend: HTML, CSS, JavaScript
-   Backend: Flask 
-   Database: SQLite / SQLAlchemy
-   Hosting: Render


## Database Design

-   Tables are designed based on the project requirements
-   Relationships between entities are maintained using keys
-   Normalization techniques are applied to reduce redundancy


## How to Run Locally

1.  Clone the repository: git clone
    https://github.com/Harshini-64-B/dbms_project.git

2.  Navigate to the project folder: cd dbms_project

3.  Install dependencies: pip install -r requirements.txt

4.  Run the application: python app.py

5.  Open in browser: http://localhost:5000
